# weed-classification
The primary contributions of this research include:
1.	Developing a new lightweight CNN-SE-Transformer architecture that maintains both the advantages of CNN's depth-wise feature learning and the efficiency required for real-world applications in precision agriculture.
2.	Providing a comprehensive analysis of how varying CNN depth affects feature learning and how residual connections improve feature learning.
3.	Introducing channel attention (SE) and transformer-based global context modeling within a single framework. Conducting extensive evaluations on multi-class weed datasets along with explanations of why certain optimizers are better than others and providing an analysis comparing optimization level performance and explanations.
Proposed results demonstrate that by combining increased CNN depth with efficient architectural design and attention mechanisms, we can greatly increase weed classification performance without increasing computational requirements, and as such will likely support the development of additional real-world precision agriculture applications
